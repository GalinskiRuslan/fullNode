Проект состоит из фронт и бек части объедененных в одном приложении.
Для развёртывания проекта важно знать, что в backend части используется prisma для работы с бд, 
после установки зависимостей нужно будет выполнить команду prisma migrate dev для миграции.
В файле .env должно быть корректно настроено DATABASE_URL, которая хранит все данные для подключения к бд
пример  DATABASE_URL="mysql://root:root@localhost:3306/nodejsprisma" где
1- тип бд 
2- имя пользователя бд 
3-пароль 
4-адрес или ip адрес с хостом и 
5-название базы.
